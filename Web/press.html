Description

Evan Todd is an independent game developer based in Columbus, Ohio.

History

Evan started work on Lemma in 2010 while attending university. The project originally began as a way to raise money for charity. Evan graduated in 2012 and worked for a small iPhone game studio for a year, continuing to develop Lemma part-time. Finally realizing Lemma would never survive as a side project, he quit in February 2014 to focus on it full-time.
